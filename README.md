## Hotkeys
| Function | Windows / Linux | Mac |
| ------ | ------ |  ------ | 
| Start Recording | F9  | Cmd+Shift+2 |
| Pause/Resume Recording | F9  | Cmd+Shift+2 |
| Stop Recording | F10  | Cmd+Option+2 |
